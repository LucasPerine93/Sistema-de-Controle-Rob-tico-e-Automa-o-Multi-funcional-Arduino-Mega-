Desenvolvimento de um sistema complexo de automação e controle robótico utilizando a arquitetura do Arduino Mega 2560. O projeto, que já soma mais de 6 meses de desenvolvimento contínuo e ultrapassa 3.000 linhas de código em C++, foi projetado para ser escalável e modular.

Principais Funcionalidades Implementadas:

Controle de Movimento: Gerenciamento preciso de 4 Motores de Passo Nema 17 e 2 Servomotores SG92R, com lógica de aceleração e posicionamento.

Sensoriamento: Integração com sensores ultrassônicos para leitura de ambiente e lógica de reação baseada em inputs externos (Medidor de Reação).

Interface Homem-Máquina (IHM): Desenvolvimento de uma interface gráfica via Monitor Serial, permitindo controle total do sistema através de menus dinâmicos, sem necessidade de reupload do código.

Documentação Integrada: Sistema inovador de manuais e "Help" embutidos na própria memória do microcontrolador, permitindo ao usuário consultar comandos e funções em tempo real.

Competências Demonstradas: Lógica de programação avançada, estruturação de código para grandes projetos, integração de hardware (sensores/atuadores) e design de interface via terminal.
